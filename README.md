# 🏥 Hospital Patient Registry System

A modern web-based **Hospital Patient Registry System** that enables hospitals and clinics to securely register, store, and manage patient records. The application features a premium glassmorphism user interface and a powerful backend built with **Node.js**, **Express.js**, and **MongoDB Atlas** for reliable cloud-based data storage.

---

## ✨ Features

* Premium Glassmorphism UI with a modern and responsive design
* Secure patient registration system
* Live MongoDB Atlas database integration
* Dynamic dashboard to view registered patient records
* Secure configuration using **dotenv**
* Fast and scalable Express.js backend

---

## 🔒 Environment Variables

Create a **.env** file in the root directory and add the following:

```env
MONGODB_URI=your_mongodb_connection_string
PORT=3000
```

**Note:** Do not upload the `.env` file to GitHub. Add it to your `.gitignore` file.

---

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/ayushkardam2004-stack?tab=repositories
```

2. Move into the project directory:

```bash
cd Hospital-Patient-Registry
```

3. Install all dependencies:

```bash
npm install
```

4. Configure the `.env` file with your MongoDB Atlas connection string.

5. Start the application:

```bash
node app.js
```

6. Open your browser and visit:

```text
http://localhost:3000
```
